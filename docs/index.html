PORTFOLIO·ARCADE
create gaming console-themed portfolio websites | one curl command to install

quick install

install portfolio-arcade with a single command:

curl -sSfL https://milxzy.github.io/portfolio-arcade/install.sh | bash

what you get

cross-platform cli tool (linux, macos, windows)
three gaming console themes (ps3 xmb, ps5 modern, nintendo wii)
modern next.js + react + typescript stack
responsive design with tailwind css
optional cms integration (decap, payload)
automatic updates
zero configuration setup

usage

after installation, create your first portfolio:

portfolio-arcade init my-portfolio

check for updates:

portfolio-arcade update

see all options:

portfolio-arcade --help

themes available

ps3 - classic playstation 3 xrossbar (xmb) interface
ps5 - modern playstation 5 user interface
wii - nintendo wii channel menu aesthetic

requirements

no dependencies - the installer handles everything
works on linux (x64/arm64), macos (intel/apple silicon), windows
generated portfolios require node.js for development
